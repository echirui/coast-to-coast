# coast-to-coast
囲碁みたいなゲーム

## Hexゲームのルール

Hexは、六角形のマスが並んだ菱形の盤を使用する2人対戦用のアブストラクトボードゲームです。

### ゲームの目的
各プレイヤーは自分の色のコマを使い、盤の向かい合った自分の色の辺同士を繋げることを目指します。先に自分の色の辺を繋げたプレイヤーの勝利となります。

### 基本的なルール
1.  **プレイヤーとコマの色**: 2人のプレイヤーはそれぞれ異なる色のコマ（一般的には赤と青）を担当します。
2.  **手番**: プレイヤーは交互に、空いている六角形のマスに自分の色のコマを1つ置きます。一度置かれたコマは動かせず、すでにコマが置かれているマスには重ねて置くことはできません。
3.  **勝利条件**: 自分の色のコマを連続して配置し、盤の向かい合う2つの辺を繋げたプレイヤーが勝利します。盤の4隅のマスは、両方の辺に属するものとみなされます。

### その他の特徴
*   **引き分けなし**: Hexは必ずどちらかのプレイヤーが勝利し、引き分けになることはありません。
*   **パイ・ルール**: 先手プレイヤーが有利であるため、公平性を保つために「パイ・ルール」が適用されることがあります。これは、先手が最初のコマを置いた後、後手が「先手と後手を交代する」か「このまま続行する」かを選択できるルールです。
*   **盤のサイズ**: 一般的な盤のサイズは11×11ですが、13×13や19×19、あるいは6×6などの小さいサイズも使用されます。
